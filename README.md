# MusicWorld Streaming Platform

MusicWorld es una aplicación web inspirada en plataformas de streaming musical como Spotify, donde los usuarios pueden reproducir canciones, visualizar videoclips y comprar música.

El sistema también permite la generación y envío automático de facturas por correo electrónico.

---

## Tecnologías utilizadas

Backend
- ASP.NET
- C#

Frontend
- HTML
- CSS
- JavaScript

Base de datos
- SQL Server

Servicios externos
- Mailtrap (sandbox para envío de correos)

---

## Características principales

- Reproducción de canciones
- Visualización de videoclips musicales
- Gestión de artistas, géneros y canciones
- Compra de canciones
- Generación de facturas
- Envío automático de facturas por correo electrónico
- Administración de catálogo musical
